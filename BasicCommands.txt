Create File:
touch file_name.txt -> for a .txt file, extension can change.

Edit File:
nano file_name.txt -> for a .txt, extension can change.
ctrl + O -> Save
ctrl + X -> Exit
return/enter -> Complete

Make Folder:
mkdir folder_name

List/Enter files:
ls -> to list
cd folder_name -> enter a folder.
cd -> exit folder.

Git Commands:
git init -> make it a repo
git add file_name.txt -> add the file to the repo
git rm --cached filename -> removes from repo
